# netcdf-leaflet-backend
Frontend for Displaying NetCDF Data in Leaflet

## Current Features:
- displaying Heightlevels dynamically
- dynamic Legends
- Timedimension Control
- GetFeatureInfo by Click Event


## Used Technology
- Librarys
    - JQuery
    - Leaflet
        - TimeDimension
            - iso8601
        - groupedLayerControl
- Source
    - CSS
        - custom Controls
